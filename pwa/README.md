# Lucid Mobile PWA (Share Target)

**Status:** Placeholder. Full scaffold lands in Sprint 2B.

This directory will hold the Lucid Progressive Web App. The PWA serves
two roles in beta:

1. **Share Target capture** - registered as a share destination on
   Android and iOS, receiving URLs from any app that supports the OS
   share sheet (capture-stage-spec.md §3.2, §5.2).
2. **Mobile UI surface** - Ask Lucid, Pending queue, Stellar View, and
   Settings, all rendered as a mobile-first PWA installed to the home
   screen.

See:
- `docs/capture-stage-spec.md` §5 - PWA capture flow
- `docs/beta-backlog.md` Sprint 2B - PWA Share Target
- `MASTER_HANDOFF.md` §6 - target directory structure

Until Sprint 2B, this directory is intentionally empty (this README aside).
